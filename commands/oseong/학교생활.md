# /oseong:학교생활 — 오성의 학교생활 도우미

## 트리거
사용자가 `/oseong:학교생활` 을 입력하면 즉시 아래 메뉴를 출력하고 선택을 기다린다.

## 실행 시 즉시 출력할 메뉴

```
📚 학교생활 도우미

무엇을 도와드릴까요?

  1. 수업노트 정리   — 클로바노트 + 슬라이드 → 노션 자동 업로드
  2. 작업현황        — 이번 학기 노션 정리 현황 확인

번호를 선택하세요.
```

선택지를 출력한 뒤 **반드시 사용자 입력을 기다린다.** 임의로 진행하지 않는다.

---

## 1번: 수업노트 정리

### 흐름
1. 아래 질문을 **한 번에** 출력한다:
   ```
   어떤 수업인가요?
   현재 등록된 수업: 서비스디자인 / 웹인터페이스설계 / 미래산업전망과스타트업 / 이미지와상상력 / 창업301 / 창업307

   몇 주차인가요?

   클로바노트 파일이 Dropbox에 저장되어 있나요? (Y/N)
   구글 슬라이드 링크가 있나요? (있으면 붙여넣기, 없으면 엔터)
   ```

2. 사용자 답변을 받은 뒤 다음 작업을 순서대로 수행한다:
   - Dropbox 경로: `C:\Users\GKL\Dropbox\02 학교 관련\26년도1학기\{수업명}\{N}주차\새로운 노트.txt` 파일 읽기
   - 파일이 크면 분할해서 읽는다 (Bash head/tail 활용)
   - 구글 슬라이드 링크가 있으면 Playwright로 접속, 각 슬라이드 캡처 후 세로 이미지 파일 저장
   - 노션에 해당 수업 서브페이지 아래 `{N}주차 강의노트` 페이지 생성
   - 강의 핵심 내용을 섹션별로 정리해 블록 추가
   - 구글 슬라이드 링크, 클로바노트 링크, 캡처 파일 경로를 함께 추가

3. 완료 후 노션 페이지 링크를 출력한다.

### 노션 페이지 구조 (기준)
```
26년도 1학기
  └── {수업명}
        └── {N}주차 강의노트
              ├── 🔗 강의자료 링크 (구글 슬라이드, 클로바노트)
              ├── 📋 수업 기본 정보 (날짜, 시간)
              ├── 📖 핵심 내용 (섹션별 정리)
              └── 🗒️ 교수님 주요 발언 / 메모
```

### 알려진 수업별 노션 페이지 ID
- 서비스디자인: `31a67569-a19a-81c6-84ba-d578bbfa056b`
- 웹인터페이스설계: `31a67569-a19a-8175-aa78-fa0c541ed6ef`
- 26년도 1학기 (최상위): `31a67569-a19a-810c-8563-eedce428eee2`
- 기타 수업: 없으면 26년도 1학기 아래 새로 생성

## 공통 원칙
- 메뉴 출력 후 **반드시 사용자 선택을 기다린다**
- 작업 중 필요한 정보가 없으면 진행하지 말고 질문한다
- 노션 API는 MCP(`mcp__notion__*`) 사용
- 슬라이드 캡처는 MCP(`mcp__playwright__*`) 사용
- 파일 읽기는 `Bash` 또는 `Read` 도구 사용
